# Rebuild Specs

This folder contains the planning artifacts for rebuilding the project before any implementation work.

## Documents

- `00-current-state.md`: Audit of the existing repository.
- `01-objectives-and-scope.md`: Draft objectives, scope boundaries, and success criteria.
- `02-technical-plan.md`: Proposed technical architecture for the new CLI-first engine.
- `03-roadmap.md`: Phased delivery plan with concrete milestones.
- `04-decisions-log.md`: Locked decisions and future non-blocking choices.

## Workflow

1. Review and adjust `01-objectives-and-scope.md`.
2. Confirm architecture choices in `02-technical-plan.md`.
3. Lock milestone priorities in `03-roadmap.md`.
4. Start implementation only after these documents are approved.
